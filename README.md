## MODULE - 1 : BUS & OPERATORS

- OPERATORS
    - Admin Can CRUD, Approve & reject
    - Bus owners can request for account

- BUS
    - Admin Can CRUD, Approve or Cancel (default in-active, active when have at least one active trip)
    - Bus owners can (add, edit, de-activate or activate)

## MODULE - 2 : STOPS

- Admin Can CRUD, Approve & reject
- Bus owner can suggest (v2)

## MODULE - 2 : ROUTES & ITS STOPS

- Admin Can CRUD, Approve & reject
- ROUTES owner can add when schedule a trip with start and end
- ITS STOPS Bus owner can suggest (v2)

## MODULE - 2 : TRIPS

- Admin Can CRUD, Approve & reject
- Bus owners can (add, edit, de-activate or activate)
